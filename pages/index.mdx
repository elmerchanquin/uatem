# Introduction
 
Welcome to UATEM (Unified Access Technology of Elements and Materials) This is the project to create a database to show the process of the creation of everything.

## What is UATEM?

A **simple**, **powerful** and **flexible** platform to know every process and requirements to create from 0 everything that humans had build.

## How to use

The documentation to use this platform is available at [https://uatem.com/usage-docs](https://uatem.com/usage-docs).
